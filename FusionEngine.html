Introduction
What is Fusion Engine and what does it include?

Fusion Engine is the RAM management and system-level optimization section in TurboFPS Pro. Unlike Perf Lab which focuses on CPU/GPU and per-game performance tuning, Fusion Engine is designed specifically to manage your device's memory, control background processes, and maintain stable performance throughout gaming sessions.

It includes four core features:

RAM Cleaner
Real-time RAM monitor with one-tap clean. Live usage bars updating every 2ms.
Real-time
DNS Manager
Custom DNS via Shizuku shell. Private DNS control without opening system settings.
System-wide
FPS Stability
Frame-pacing optimization toggle to reduce drops and smooth out gameplay.
Toggle
RAM Booster
Background app killer to aggressively free up memory for your game.
One-tap
Shizuku Required
Fusion Engine uses Shizuku for shell-level access. Root is NOT required. Make sure Shizuku is running before using any toggle features.

3.2 DNS Manager

The DNS Manager lets you view and change your device's Private DNS setting without opening Android system settings. It reads and writes the private_dns_mode global setting via Shizuku shell.

DNS status display values:

Private DNS: OFF
DNS is not encrypted. Default state on most devices.
Automatic
Opportunistic DNS-over-TLS is active.
Hostname Shown
Custom DNS provider is configured (e.g. dns.google).
Shizuku Not Running
Shizuku is unavailable — status cannot be read.
Takes Effect Immediately
Changing DNS via Fusion Engine takes effect system-wide immediately. It is equivalent to going to Settings > Network > Private DNS and typing the hostname manually.
3.3 FPS Stability

FPS Stability is a toggle that applies frame-pacing optimization settings via Shizuku. When enabled, it adjusts system-level parameters to reduce frame drops and smooth out gameplay.

FPS Stability
Writes fps stability flag to SharedPreferences (key: fps / enabled). Applies Shizuku shell commands to optimize frame scheduling. State persists — setting is restored on app relaunch.
Safe
Samsung Devices — Do Not Use
Samsung's OneUI has its own frame management layer which can conflict with FPS Stability commands. Enabling on Samsung may cause stutters instead of improvement.
3.4 RAM Booster

RAM Booster is a one-tap background app killer. When toggled on, it signals the system to aggressively trim background processes and free up memory for the foreground app (your game).

RAM Booster
Uses Shizuku shell to send memory trim signals to background processes. Kills non-essential background services to free up RAM. Works best when triggered right before launching a game. State is saved — re-enables automatically on next app open.
Kills BG
Shizuku Required
RAM Booster requires Shizuku to be running. If Shizuku is not active when you toggle it, the switch will revert and a toast message will appear: "Shizuku is not running".

Usage Guide
Setup, daily routine, and DNS recommendations
4.1 First-Time Setup
  • 1
    Install Shizuku from the Play Store or GitHub.
  • 2
    Open Shizuku and start it via Wireless ADB or USB ADB.
  • 3
    Open TurboFPS Pro and navigate to the Fusion Engine tab.
  • 4
    Grant Shizuku permission when prompted on first use of any toggle.
  • 5
    RAM stats should now display automatically with live updates.
4.2 Before Gaming — Recommended Routine
  • Check Used RAM — if above 70% of Total RAM, tap Clean in the RAM Cleaner.
  • Toggle RAM Booster ON to free up background memory.
  • Enable FPS Stability (non-Samsung devices only).
  • Pull down to refresh DNS status and verify your DNS is set correctly.
  • Launch your game — system is now optimized.
Pro Tip — Fresh Reboot
For best results, use Fusion Engine after a fresh reboot. A rebooted device with RAM Booster enabled typically gives 200–400 MB more free RAM before game launch.
4.3 DNS Recommendations for Gaming

Custom DNS can reduce ping and improve connection stability in online games. Recommended providers:

Troubleshooting
Common issues and how to resolve them
"Shizuku is not running" toast appears
Open the Shizuku app and tap Start. Re-enable the toggle after Shizuku is running.
RAM stats not updating
Check if the Fragment is visible. Swipe down to refresh. Restart the app if needed.
DNS status shows "Unknown"
Pull to refresh. If persistent, toggle Airplane mode off/on to reset the network stack.
FPS Stability causing stutters
Disable the toggle — this is expected behavior on Samsung devices. See Section 3.3.
RAM Booster not freeing memory
Some OEMs lock background kill behavior. Try using the Clean button in RAM Cleaner instead.
Switch reverts immediately
Shizuku permission was denied. Open Shizuku > Grant Permission and try again.

Safety & Best Practices
Non-destructive, reversible, and root-free

Fusion Engine is designed to be non-destructive and reversible. All changes made by toggles can be undone by simply turning them off. No permanent system modifications are made.

Don't Force-Close Shizuku While Active
If Shizuku is killed while features are active, toggles may get stuck in the ON state.
Close Important Apps Before RAM Booster
RAM Booster will kill some background apps. Close important apps manually before enabling it.
DNS Changes Are System-Wide
Games and all other apps on your device will use the new DNS provider — not just TurboFPS.
All Features Revert When Toggled OFF
All features revert to safe defaults when toggled OFF. No permanent changes are ever made.
Root-Free
Fusion Engine does NOT require root access. No system files are modified. All commands run via Shizuku's ADB-level permission — the same level Android Debug Bridge uses.
