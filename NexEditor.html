What is Nex Editor?
Overview & purpose

Nex Editor is a built-in feature inside TurboFPS Pro that lets you unlock and change the FPS and graphics settings of BGMI / PUBG Mobile beyond what the game normally allows.

Most phones are locked to 60 FPS or lower inside the game. With Nex Editor, you can push your device to 90 FPS or even 120 FPS — giving you smoother gameplay and a real competitive advantage.

🎮
Supported Games
BGMI · PUBG Mobile Global · PUBG Mobile KR · PUBG Mobile VN · PUBG Mobile TW

Requirements
Before you begin

Before using Nex Editor, make sure the following are in place:

RequirementWhat it does
ShizukuMust be installed and running. Nex Editor uses Shizuku to access game files safely without root.
BGMI / PUBG MobileThe game must be installed on your device. Nex Editor reads settings from the installed game.
TurboFPS ProYou need TurboFPS Pro installed. Nex Editor is found inside the app under its own tab.
Shizuku PermissionTurboFPS Pro needs Shizuku permission to load and save your game settings.
ℹ️
Don't have Shizuku?
Download Shizuku for free from the Play Store. Open Shizuku and follow the setup steps. Once it shows "Running", come back to TurboFPS Pro.
3
How to Open Nex Editor
Navigation inside TurboFPS Pro
  • Open TurboFPS Pro on your phone.
  • At the bottom navigation, tap the Nex Editor tab (looks like a file/editor icon).
  • The Nex Editor screen will open with tabs at the top: FPS, Graphics, Style, Profile, Advanced, and Presets.
  • The app will automatically load the built-in default file. If it says "Open a SAV file to begin", your game may not be installed or Shizuku may not be running.
⚠️
Important
Make sure BGMI / PUBG Mobile is installed before opening Nex Editor. Shizuku must be running in the background for the auto-load to work.

5
Mode Selector
Both / Combat / Lobby

Just below the game chips, you will see three buttons: Both, Combat, and Lobby. This controls which game mode your settings apply to.

ModeWhat it does
BothChanges apply to both Battle (Combat) and Lobby at the same time. This is the default and recommended setting.
CombatChanges only apply to Battle / Combat mode. Your Lobby settings stay the same.
LobbyChanges only apply to Lobby (main menu area). Battle settings stay untouched.
💡
Tip
Use Both mode for quick setup. Use Combat or Lobby separately if you want different FPS or quality for each situation.

FPS Tab
Unlocking frame rate

The FPS tab is where you unlock and set the frame rate for your game. This is the most important tab in Nex Editor.

6.1 FPS Presets

You will see 8 preset buttons. Tap any one to instantly set that frame rate.

6.2 Custom FPS Slider

Below the presets, there is a slider you can drag to set a custom FPS value. Use this if none of the presets match exactly what you want.

6.3 Fine-Tune Switch

There is a Fine-Tune toggle switch below the slider. Turning this ON enables custom FPS slider support in the app UI, linked to your current loaded file.

⚠️
Device Warning
Setting FPS higher than what your device supports can cause overheating or instability. Start with 60 FPS first and test before going to 90 or 120. If the game becomes laggy or crashes, lower the FPS value.

Graphics Tab
Visual quality settings

The Graphics tab lets you control the visual quality of the game — separately for Battle and Lobby if needed.

FeatureWhat it does
Battle QualitySet the render quality during actual matches. Lower = better performance. Higher = better visuals.
Lobby QualitySet render quality in the lobby / main menu area.
Lobby Enhancement ModeA toggle switch to reduce lobby visuals for battery saving.
BrightnessA slider to control screen brightness inside the game. Adjust to your comfort.
💡
Pro Tip
For competitive play, lower Graphics quality + higher FPS is the better combination. Save battery by reducing Lobby quality since the lobby is not gameplay.
8
Style Tab
Render style & environment look

The Style tab controls the visual rendering style of the game — the overall look and color style of the environment. You can set the style separately for Battle mode and Lobby mode.

ℹ️
Note
Render style is a visual preference and does not directly affect FPS or performance. Experiment with different styles to see which one you like most.

10
Advanced Tab
Power users only

The Advanced tab shows all remaining game settings not shown in the other tabs. These are raw setting values from your game file. You can search for a specific setting using the search bar at the top.

⚠️
Caution — Advanced Tab
Only change values here if you know exactly what the setting does. Changing unknown values can break your game settings and require a reinstall. Stick to the other tabs (FPS, Graphics, Style, Profile) for safe changes.
11
Presets Tab
Save your configurations

The Presets tab lets you save your current settings as a named preset so you can quickly load them again later.

FeatureWhat it does
Save PresetTap the + button (FAB) to save your current settings with a name.
Load PresetTap any saved preset card to apply those settings instantly.
Edit / DeleteLong-press a preset card or tap the edit icon to rename or delete it.
SearchUse the search bar at the top to find a preset by name.
💡
Tip
Create multiple presets — for example: "90 FPS Competitive", "60 FPS Battery Saver", "Max Quality Lobby". Switch between them instantly without reconfiguring everything manually.
12
Applying Your Settings
Save & write changes to your game

After making changes in any tab, you need to apply them to save them to your game. Look for the floating action button (FAB) in the bottom-right corner of the screen.

✔️
Apply
Writes your changes to the game settings file. Do this after every change.
💾
Save
Saves a copy of the current settings file to your phone's storage.
📂
Open
Opens file picker to load a SAV file from your internal storage.
📥
Import
Import directly from game — no manual file selection needed.
🔄
Reset
Resets all settings to game's original defaults. Not available in Beta state.
⚠️
Always Apply!
Changes are NOT saved automatically. After adjusting FPS or any setting, always tap Apply to make the changes take effect. Then restart BGMI / PUBG Mobile for the new settings to load in the game.
13
Quick Start
Unlock FPS in 5 steps

Follow these steps to unlock FPS on your device right now:

  • Make sure Shizuku is running. Open the Shizuku app and confirm it shows "Running".
  • Open TurboFPS Pro and go to the Nex Editor tab.
  • Tap your game chip at the top (e.g. BGMI). The settings will load automatically.
  • Go to the FPS tab and tap the FPS preset you want (e.g. 90 for 90 FPS).
  • Tap Apply (the floating button at bottom right). Done — restart BGMI and enjoy!
✅
You are all set!
